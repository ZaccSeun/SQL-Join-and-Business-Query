# SQL-Join-and-Business-Query

## Introduction

This project is about joining different tables in SQL and taking decisons using business query

## Project Overview
Five (5) tables were joined together using left join to form a comprehensive mastertable
The comprehensive table was used for analysis and the following were calculated;
- Top 20 customers based on total sales and total quantities.
- Top 20 sales region that generated top rated customers
- Top 20 products based on total sales and total quantity
- Overall best product category
- Year on Year growth

## Conclusion
The tables were joined and the business query carried out appropriately.
